This task has two agent types: The Master Scheduling Agent and the Car Agent. 
The Car Agent starts by requesting to be scheduled by the Master Scheduling Agent.
The Master Scheduling Agent then looks at the request, the Car's preferences and responds accordingly. 
If the Master can schedule the Car Agent, it will agree. If it can't, it will refuse. 
The Car Agent can reply with a Disconfirm or Confirm upon hearing this.
